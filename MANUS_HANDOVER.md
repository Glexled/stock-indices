# A股指数对比项目接力指南 (Manus Handover)

## 1. 项目概况
这是一个专业级的 A 股指数对比 Web 应用，旨在为用户提供类似同花顺（THS）风格的 9 大主要指数实时与历史趋势对比工具，优化了移动端体验，并部署于 Vercel。

## 2. 已实现的核心功能
- **9 大指数覆盖**：上证指数、上证 50、沪深 300、中证 500、中证 1000、中证 2000、科创综指、科创 50、创业板指。
- **专业数据源**：已从腾讯接口切换至 **东方财富 (Eastmoney) 专业行情接口**，支持 5 秒实时刷新。
- **智能交互优化**：
  - **动态排序提示框**：鼠标或手指滑动时，数据框内的指数按该时间点的**涨幅从高到低自动排序**。
  - **市值排序卡片**：下方指数切换卡片严格按市值从大到小排列。
  - **默认全显**：默认显示所有 9 条曲线，点击卡片可隐藏/显示。
  - **加载动画**：解决了 Vercel 冷启动导致的白屏等待焦虑。
- **部署环境**：适配 Vercel Serverless Functions，代码托管于 GitHub 仓库 `Glexled/stock-indices`。

## 3. 技术细节
- **前端**：原生 JS + ECharts 5.4.3 + CSS3 响应式设计（适配暗黑模式）。
- **后端**：Python Flask (Vercel Serverless)，路径为 `/api/minute`。
- **配置文件**：`vercel.json` 处理路由重定向。
